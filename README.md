# FairSplit 🏠💰

A simple web application for roommates to split bills and track who owes who. Built entirely with frontend technologies.

## 🚀 Live Demo
🔗 [Click here to view the live website](https://RahulPrasad101-7.github.io/FairSplit/)

---

## 🛠️ Features
* **Split Bills:** Input a bill amount, select who paid, and list all the roommates sharing the expense.
* **Track Balances:** Automatically calculates and displays who is owed money (green cards) and who owes money (red cards).
* **Instant Payback:** Includes a "Pay Debt" button that clears roommate balances directly.
* **Data Persistence:** Uses browser `LocalStorage` so your bills don't disappear when you refresh the page.

---

## 💻 Built With
* **HTML5** - Structure and layout
* **CSS3** - Custom styling, responsive cards, and clean buttons
* **JavaScript** - Dynamic DOM updates and mathematical balance calculations

---

1. Download or clone this repository.
2. Open the `index.html` file directly in any web browser.
